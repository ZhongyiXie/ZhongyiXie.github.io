五、数学秒杀技巧（5个必考高频考点）

技巧1：工程/行程/浓度题 —— 设最小公倍数

原理：把总量设为各效率数字的最小公倍数，避开分数计算。

例：一项工程，甲单独做15天完成，乙单独做20天完成。甲乙合作3天后，乙离开，甲单独完成剩下的。问还要几天？

秒杀法：设总量=60（15和20的最小公倍数）→ 甲效率4，乙效率3 → 合作3天完成(4+3)×3=21 → 剩余39 → 39/4=9.75天 ✅

速度提升：3倍。省去所有分数通分计算。

技巧2：排列组合 —— 先选后排 + 相邻打包

总口诀：先选后排，特殊优先，相邻打包，不相邻插空

例1：6人中选3人站一排，甲必须在中间 → C(5,2)×2! = 20种
例2：5人站一排，甲乙必须相邻 → 打包×排列 = 4!×2! = 48种
例3：5人站一排，甲乙不能相邻 → 全排-相邻 = 120-48 = 72种

技巧3：概率 —— "至少"题一律反向算

口诀：遇"至少"→ 1减去反面

例：投篮3次，命中率0.6，至少命中1次的概率？
秒杀：1 - 0.4³ = 0.936 ✅（省去P(1)+P(2)+P(3)的麻烦）

技巧4：数列 —— 优先看差值规律

管综数列题 80% 差值法可秒杀

例：2, 5, 10, 17, 26, ? → 差值=3,5,7,9 → 下个差11 → 37 ✅
口诀：做差→看规律→再差→等差/等比→反推
注意：一次差看不出规律就做二次差——管综最常见的数列就是二阶等差

技巧5：不定方程 —— 消元法

管综每年必出1道，送分题

例：铅笔1元+钢笔3元=20支共44元 → x+y=20, x+3y=44 → y=12, x=8
核心：不是猜数，是直接消元解方程，10秒出答案

6.1 形式逻辑公式表（占60%逻辑题 —— 必须满分的部分）

逻辑类型公式示例注记
如果A则B A→B
等价于：¬B→¬A		 "如果下雨则地湿"
= "地不湿→没下雨"		 逆否命题，管综最常考
只有A才B B→A "只有考过才能毕业"
= "毕业→考过"		 方向反过来！和"如果则"容易混
除非A否则B ¬A→B "除非下雨，否则出门"
= "不下雨→出门"		 = "如果不下雨就出门"
三段论 A→B, B→C → A→C "安全专家→懂密码，懂密码→懂数学"
→ "安全专家→懂数学"		 铁律：中项必须周延至少一次
有的互换 有的A是B = 有的B是A "有的安全专家是博士"
= "有的博士是安全专家"		 "所有A是B"不能倒过来

6.2 论证逻辑解题框架（你的售前优势区）

核心心法：读题时问自己——"这段话的漏洞在哪？"和你15年写方案、找别人方案漏洞的思维一模一样。
题型问法标志解题方向
削弱题 "最能削弱上述论证" 找因果倒置（B导致A而非A导致B）

检验法：假设不成立→论证就垮→这就是答案
